# 壁纸网站使用说明

本项目已改造为一个功能完整的壁纸展示网站，参考了 haowallpaper.com 的设计理念。

## 功能特性

### ✅ 已实现功能

1. **壁纸展示**
   - 响应式网格布局（瀑布流）
   - 支持 PC 端、移动端、头像等多种设备类型
   - 图片懒加载优化
   - 悬停效果和动画

2. **搜索与筛选**
   - 按文件名搜索
   - 按设备类型筛选
   - 按分类筛选
   - 按标签筛选
   - 多条件组合筛选

3. **壁纸详情页**
   - 高清图片预览
   - 详细信息展示（分辨率、分类、标签等）
   - 下载功能
   - 分享功能
   - 预览次数统计

4. **用户体验**
   - 深色模式支持
   - 响应式设计（移动端适配）
   - 加载状态提示
   - 错误处理
   - 平滑动画过渡

## 项目结构

```
app/
├── components/
│   ├── WallpaperCard.vue      # 壁纸卡片组件
│   └── WallpaperGrid.vue       # 壁纸网格组件
├── composables/
│   └── useWallpaper.ts        # 壁纸相关的 composable
├── pages/
│   ├── index.vue              # 首页（壁纸列表）
│   └── wallpaper/
│       └── [id].vue           # 壁纸详情页
└── api/
    └── generated/              # 自动生成的 API 客户端
```

## 使用说明

### 1. 启动开发服务器

```bash
bun run dev
```

访问 `http://localhost:3000` 查看网站。

### 2. 首页功能

- **搜索框**：输入文件名进行搜索
- **设备类型筛选**：选择 PC 端、移动端或头像
- **分类筛选**：输入分类名称
- **标签筛选**：选择多个标签进行筛选
- **加载更多**：点击按钮加载更多壁纸

### 3. 壁纸卡片

- **点击卡片**：跳转到详情页
- **悬停显示**：显示下载和查看按钮
- **分辨率标签**：右上角显示图片分辨率
- **统计信息**：显示预览次数和分类

### 4. 详情页功能

- **高清预览**：查看完整尺寸图片
- **下载原图**：下载原始图片文件
- **分享功能**：分享壁纸链接
- **标签导航**：点击标签跳转到筛选页面

## 技术栈

- **框架**：Nuxt 4
- **UI 组件库**：Nuxt UI
- **样式**：Tailwind CSS
- **图片处理**：Nuxt Image
- **类型安全**：TypeScript
- **API 客户端**：自动生成的 OpenAPI 客户端

## 自定义配置

### API 地址配置

在 `.env` 文件中配置：

```env
API_BASE_URL=http://127.0.0.1:8203
```

或在 `nuxt.config.ts` 中配置。

### 页面布局

修改 `app/app.vue` 自定义头部和底部。

### 样式定制

修改 `app/assets/css/main.css` 自定义样式。
